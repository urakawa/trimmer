# エディタでjsonを書いてmustacheテンプレートを読み込むとコンパイルしたhtmlを出力するツール

## 使い方

### 起動
まだバイナリはリポジトリに入れていないです
ので、実行するには以下の手順が必要です

- nodeとelectronをインストール
- srcディレクトリで npm install
- srcディレクトリで electron .

### エディタ
レンダリングに使用するjsonを編集します

「json読込」「json保存」ボタンで読み込み、保存が可能です。

### mustacheレンダリング
mustacheテンプレートに編集した変数を反映させ、htmlとして出力します。

「テンプレ変換」ボタンをmustacheテンプレートを選択すると、変数が反映されたhtmlが出力されます。ファイル名は元のテンプレートのファイル名の末尾に ".rendered.html" がついた名前になります。
